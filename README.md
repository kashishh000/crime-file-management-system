# Crime File Management System

## 📌 Description
A web-based application designed to digitize and streamline the process of filing and managing crime complaints. The system enables users to register complaints online and allows administrators to monitor, update, and manage records efficiently.

## 🚀 Features
- User registration and login system  
- Online complaint filing  
- Complaint status tracking  
- Admin dashboard for managing complaints  
- Victim and suspect information handling  

## 🛠 Technologies Used
- Java (Core Java / Servlets concept)
- JSP (conceptual)
- HTML, CSS (UI design concept)
- MySQL (database concept)

## ⚙️ System Architecture
The system follows a three-tier architecture:
- **Presentation Layer** – User interface for interaction  
- **Application Layer** – Backend logic handling requests  
- **Database Layer** – Stores complaint and user data
